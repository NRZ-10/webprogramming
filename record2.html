HTML LISTS

ORDERED LIST

Displays items in a specific order (numbered).Each item is marked with numbers, letters, or Roman numerals.

  1. Numeric
  2. Capital Roman Number
  3. Small Roman Number
  4. Capital Alphabets
  5. Small Alphabets
Example
  1. First Item
  2. Second Item
  1. First Item
  2. Second Item

UNORDERED LIST

Displays items with bullet points (no specific order).Commonly used for menus and simple item groupings.

  • Disc
  • Circle
  • Square
  • None
Example
  • Apple
  • Banana
  • Mango

DESCRIPTION LIST

Used for terms and their descriptions.Contains dt (definition term) and dd (definition description).
